widget_recipe_properties

This message allows you to send login details and skip login with google (or some other method).
This method is suitable if you already have the user's email.
The event looks like this:
{ 'name' : 'widget_login_details' , 'data' : { 'userId' : $routeParams.userId, 'email': $routeParams.email, 'name' : $routeParams.firstName, 'lastName' : $routeParams.lastName } }
